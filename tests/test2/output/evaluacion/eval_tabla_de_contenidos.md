# Evaluación: Tabla de contenidos

## Análisis
El documento incluye el encabezado `## Tabla de contenidos`, pero inmediatamente después no se presenta ningún índice, lista de secciones, hipervínculos internos ni numeración. El espacio permanece completamente vacío hasta el siguiente apartado (`## Registro de cambios`). Aunque existe el título formal, la ausencia total de contenido, estructura jerárquica y enlaces de navegación hace que, a efectos prácticos, el documento **no disponga de una tabla de contenidos funcional**. No se evidencia ningún intento de generación automática ni manual, por lo que no cumple con los requisitos de estilo ni de utilidad esperados para este criterio.

## Puntuación
**Puntuación:** 0/10

## Observaciones
- Generar la tabla de contenidos de forma automática utilizando las herramientas nativas del editor (Word, Google Docs, LaTeX o extensiones de Markdown como `markdown-toc` o `[[TOC]]`).
- Asegurar que incluya todos los niveles de encabezados presentes en el documento (H1, H2, H3) con hipervínculos clicables que dirijan directamente a cada sección.
- Si el documento se exporta a PDF o se imprime, añadir números de página alineados a la derecha para facilitar la referencia física.
- Configurar la tabla para que se actualice automáticamente ante cambios estructurales, evitando así desactualizaciones o enlaces rotos en futuras versiones.
